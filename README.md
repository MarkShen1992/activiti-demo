# activiti-demo
> 工作流学习

本项目的是学习工作流引擎 [activiti](https://www.activiti.org/), 本项目使用的是 activiti6.0。

## 画图工具

本项目使用 eclipse 中的 activiti 画图插件来制作工作流，然后再导入到 IdeaJ 中使用。

### [插件安装][1]

**Name**: Activiti BPMN 2.0 designer

**Location**: http://activiti.org/designer/update/
